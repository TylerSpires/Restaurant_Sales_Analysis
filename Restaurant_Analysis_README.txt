Restaurant Sales Analysis Project

Objective:
Analyze restaurant sales data to identify top-selling menu items, category performance, and revenue trends.

Tools Used:
- PostgreSQL
- SQL
- Excel

Skills Demonstrated:
- Data cleaning
- SQL joins
- Aggregations
- KPI development
- Dashboard creation
- Data visualization

Business Questions Answered:
1. What was the total revenue?
2. Which categories generated the most revenue?
3. Which items were ordered most frequently?
